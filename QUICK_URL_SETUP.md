# Quick Start: Simplify Your URL

## Fastest Solution (Free, No Domain Purchase)

1. **Go to Vercel Dashboard:**
   - Visit [vercel.com](https://vercel.com)
   - Select your resume project

2. **Change Project Name:**
   - Settings → General → Project Name
   - Change to: `randal-resume` or `randal-portfolio`
   - Save changes

3. **Your new URL will be:**
   - `randal-resume.vercel.app`
   - Much simpler than `Randal.devtest512.info`!

4. **Remove old domain (optional):**
   - Settings → Domains
   - Find `Randal.devtest512.info`
   - Click remove

## Done! ✅

Your site will now be accessible at the simpler URL. No code changes needed.

---

## Want a Custom Domain Instead?

For a custom domain like `randal.dev`, see the full [DOMAIN_SETUP.md](./DOMAIN_SETUP.md) guide.

**Quick Steps:**
1. Purchase domain (~$20-30/year for .dev)
2. Add domain in Vercel → Settings → Domains
3. Configure DNS records at your registrar
4. Wait 5-30 minutes for DNS propagation
5. Done!

**Recommended domains:**
- `randal.dev` - Clean and professional
- `randalderego.com` - Full name
- `rderego.com` - Abbreviated
